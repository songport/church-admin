🎯 배포 순서
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Step 1️⃣  MongoDB Atlas 설정 (5분)
  ├─ 계정 생성
  ├─ 클러스터 생성
  ├─ 사용자 생성
  └─ 연결 문자열 복사

  Step 2️⃣  .env 파일 작성 (2분)
  └─ MONGODB_URI, JWT_SECRET 입력

  Step 3️⃣  GitHub 푸시 (3분)
  ├─ git add .
  ├─ git commit
  ├─ git remote add
  └─ git push

  Step 4️⃣  Vercel 배포 (3분)
  ├─ 대시보드 접속
  ├─ 프로젝트 선택
  ├─ 환경 변수 설정
  └─ Deploy

  Step 5️⃣  Render 배포 (5분)
  ├─ 대시보드 접속
  ├─ Web Service 생성
  ├─ 환경 변수 설정
  └─ Deploy

  Step 6️⃣  최종 테스트 (2분)
  └─ 웹앱 접속 및 기능 확인

  ⏱️  총 소요시간: 15-20분
